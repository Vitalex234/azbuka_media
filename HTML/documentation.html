Getting Started

RepairEd is a HTML/CSS template based on Bootstrap framework. It a clean, responsive & easy-to-use template, specialize in Digital Repair (Computer Repair, Mobile Repair, Desktop and Laptop Repair, SmartPhones Repair, etc...). Repaired include shop template for sale device and hardware.

General Markup

This is the base of each HTML file: 

<!DOCTYPE html>
<html lang="en">
<head>
	head content (Meta tags, JS/CSS files (main/vendor))
</head>
<body>
	<div class="main-header-wrapper">
		<header class="main-header">
			header content
		</header>
	</div>
	<div class="modal-wrapper">
			modal wrapper
	</div>
	<div class="content-wrapper">
			content-wrapper
	</div>
	<div class="content-wrapper">
			content-wrapper
	</div>
</body>

Features

RepairEd have a more features, include:

  1. Modren & Clean design
  2. Smooth scroll animation
  3. Valid CSS
  4. Cross Browser optimize
  5. Many variants of Home Page
  6. MailChimp Integration
  7. Working Ajax Contact/Appointment form
  8. Google Web Font
  9. Google Map
  10. Ionic Icons and Fonw Awesome Font Pack
  11. Blog/Gallery/Shop pages variants
  12. and more...
